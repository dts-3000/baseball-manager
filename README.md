# ⚾ Franchise GM 2026

MLB franchise management game with real 2026 Opening Day rosters for all 30 teams.

## Features
- Real 2026 MLB rosters (Opening Day lineups + rotations)
- 162-game schedule with live play-by-play game engine
- Pitcher fatigue & bullpen management
- Trade offers & free agent market
- Amateur draft & minor league farm system
- Full standings, playoffs & multi-season franchise mode
- Player aging & development

## Run locally

```bash
npm install
npm start
```

## Deploy to Vercel

Push to GitHub and import the repo at [vercel.com](https://vercel.com).  
Framework preset: **Create React App** — no other config needed.
